# SaaS Authentication & Payment API

Production-ready SaaS boilerplate with JWT auth, OAuth login, and payment gateway integration.

## 🚀 Live Demo
- Frontend: https://saas-authentication-payment-api.vercel.app
- API Docs: https://saas-authentication-payment-api-production.up.railway.app/api-docs

## 🛠 Tech Stack
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL + Prisma ORM
- **Auth**: JWT, bcrypt, Passport.js (Google + GitHub OAuth)
- **Payment**: Midtrans
- **Frontend**: React + Vite + Tailwind CSS
- **Deployment**: Railway (Backend + DB), Vercel (Frontend)

## ✨ Features
- JWT Authentication (Access + Refresh Token)
- OAuth Login (Google + GitHub)
- Payment Gateway Integration (Midtrans)
- Webhook Handling
- Transaction Management
- API Documentation (Swagger)
